# คู่มือการติดตั้งระบบ Vipat Hotel (Installation Guide)

เอกสารนี้สำหรับฝ่ายไอทีหรือผู้ดูแลระบบ เพื่อนำ Source Code ไปติดตั้งบนเครื่องเซิร์ฟเวอร์หรือคอมพิวเตอร์ใช้งานจริง

---

## สิ่งที่ต้องเตรียม (Prerequisites)
เครื่องคอมพิวเตอร์ที่จะใช้รันระบบ ต้องติดตั้งโปรแกรมดังนี้:
1. **Node.js** (เวอร์ชั่น 18 ขึ้นไป) - ดาวน์โหลดได้ที่ [nodejs.org](https://nodejs.org)
2. **Git** - สำหรับดึงโค้ด (Optional)

---

## ขั้นตอนการติดตั้ง (Installation)

1. **ดาวน์โหลดไฟล์:**
   นำโฟลเดอร์โปรเจกต์ `Vipat-hotel` ไปวางไว้ที่เครื่องคอมพิวเตอร์ที่ต้องการใช้งาน (เช่น เครื่องหน้าเคาน์เตอร์ หรือ Server)

2. **ติดตั้ง Library:**
   เปิด Command Prompt (CMD) หรือ Terminal เข้าไปที่โฟลเดอร์โปรเจกต์ แล้วพิมพ์คำสั่ง:
   ```bash
   npm install
   ```
   *รอจนกว่าจะติดตั้งเสร็จ (ห้ามปิดหน้าจอ)*

3. **ทดสอบรันระบบ:**
   พิมพ์คำสั่ง:
   ```bash
   npm run dev
   ```
   ระบบจะแจ้งว่ารันอยู่ที่ URL ไหน (ปกติคือ `http://localhost:5173`)
   ให้เปิด Google Chrome เข้าไปที่ลิงก์นั้น

---

## การนำไปใช้งานจริง (Production Build)

หากต้องการความเสถียรและเร็วสูงสุด แนะนำให้ทำขั้นตอน "Build" ดังนี้:

1. **สร้างไฟล์สำหรับ Production:**
   ```bash
   npm run build
   ```
   ระบบจะสร้างโฟลเดอร์ชื่อ `dist` ขึ้นมา

2. **การรันไฟล์ Production:**
   คุณสามารถใช้โปรแกรมเช่น `serve` เพื่อรันโฟลเดอร์ `dist`:
   ```bash
   npx serve -s dist
   ```
   จากนั้นเข้าใช้งานได้ผ่าน Browser

---

## การตั้งค่าเพิ่มเติม (Configuration)

### การเชื่อมต่อเครื่องพิมพ์
- ระบบใช้ฟังก์ชันการพิมพ์มาตรฐานของ Browser (`window.print()`)
- ให้ตั้งค่าเครื่องพิมพ์ (Printer) ในคอมพิวเตอร์ให้เป็น Default
- เมื่อกดปุ่ม "พิมพ์" ในโปรแกรม หน้าต่างพิมพ์ของ Windows/Mac จะเด้งขึ้นมาเอง

### การใช้งานหลายเครื่อง (LAN)
- หากต้องการให้เครื่องอื่นในวง LAN เข้าใช้งานได้
- ให้รันด้วยคำสั่ง `npm run dev -- --host`
- เครื่องอื่นสามารถเข้าผ่าน IP Address ของเครื่องแม่ เช่น `http://192.168.1.100:5173`

---

*หากติดปัญหาการติดตั้ง ให้ตรวจสอบไฟล์ `package.json` ว่ามี Dependencies ครบถ้วนหรือไม่*
